# Obsidian Config Template

Every Obsidian vault stores its settings in a `.obsidian` folder at the root of the vault. The contents of this directory can be replaced with the files here to apply a consistent base configuration to any vault.

## Usage

Copy the contents of this folder into the `.obsidian` folder of your vault:

```sh
cp -r path/to/configs/obsidian/. /path/to/your/vault/.obsidian/
```

> The `.obsidian` folder is hidden in macOS by default. Use `Cmd+Shift+.` in Finder or `ls -a` in the terminal to see it.
